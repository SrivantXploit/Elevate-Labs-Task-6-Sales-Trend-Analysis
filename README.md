# Elevate-Labs-Task-6-Sales-Trend-Analysis

## Objective
Analyze monthly sales trends using SQL aggregate functions in MySQL Workbench.

## Tools Used
- MySQL Workbench
- SQL
- Microsoft Excel (Dataset Creation)
- GitHub

## Dataset Details
The dataset contains:
- Order ID
- Order Date
- Sales Amount
- Product ID

A custom sales dataset was created containing 16 sales records from January 2026 to August 2026.

## SQL Operations Performed
- Created a database named `sales_analysis`
- Created the `online_sales` table
- Inserted sales records into the table
- Calculated monthly total revenue
- Calculated total monthly orders
- Filtered sales data between specific months
- Retrieved the top 3 months based on total revenue
- Sorted and grouped sales data using SQL aggregate functions

## SQL Concepts Used
- SELECT
- SUM()
- COUNT()
- GROUP BY
- ORDER BY
- WHERE
- BETWEEN
- YEAR()
- MONTH()
- LIMIT

The project successfully:
